I Interview Prep Bot
An intelligent full-stack web application that helps users prepare for technical interviews by simulating real-time question/answer sessions, tracking progress, and offering personalized feedback.

📌 Features
🔐 User Authentication (Signup / Login)
📝 AI-Powered Interview Simulation
Domain-based question generation (React, Python, etc.)
Answer evaluation with dynamic feedback
📊 Progress Tracking Dashboard
Tracks scores, sessions, time spent
Achievements & top performance stats
👤 Profile Management
Avatar, bio, preferred domains
📁 MongoDB Storage for users, sessions, and feedback
⚡️ Cloudinary Support for avatar uploads
🔐 JWT-based auth with protected routes

1. Clone the repo
git clone https://github.com/Shaunak-bit/Ai-interview-prep-bot.git
cd ai-interview-prep-bot

2. Setup Backend
cd server
npm install
cp .env.example .env  # Fill in your Mongo URI, JWT secret, etc.
node index.js

3)Setup Frontend
cd ../client
npm install
npm run dev
